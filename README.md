## Workflow

This challenge was solved using a skill-driven, AI-assisted workflow:

1. **PRD generation** — Used the [`to-prd`](https://github.com/mattpocock/skills/blob/main/to-prd/SKILL.md) and [`grill-me`](https://github.com/mattpocock/skills/blob/main/grill-me/SKILL.md) skills to draft and refine [`PRD.md`](./PRD.md) from the challenge brief.
2. **Issue breakdown** — Used the [`to-issues`](https://github.com/mattpocock/skills/blob/main/to-issues/SKILL.md) skill to decompose the PRD into discrete GitHub issues.
3. **Sequential implementation** — Each issue was implemented one at a time by a fresh AI agent, following a Red–Green–Refactor (TDD) loop per issue.
4. **Bug sweep** — A final AI pass reviewed the codebase to surface and fix any remaining bugs.
